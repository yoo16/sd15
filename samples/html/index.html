Formタグ

form

入力・送信フォームを作成する際に使用

form タグ内で入力されたデータをWebサーバに送信

form を利用しなくても送信する方法はあるが、JavaScrpit が必要となる

送信先は action 属性、転送方法 method属性で指定

  • formタグの中に Formコントロールタグ記述
  • action に送信先URLを指定
  • method に送信方式を指定

Formコントロールタグ

input

  • 入力の基本タグ
  • name: Webに送信する項目名(key)
  • value: nameに対する値(value)
  • type: inputの種別 HTML5で種類が増え、ブラウザが対応していれば認識

text

  • テキスト入力
  • name をキーとして 入力した値が value として送信される
  • デフォルト値を設定したい場合は、value を記述する

password

  • パスワード入力
  • name をキーとして 入力した値が value として送信される
  • デフォルト値を設定したい場合は、value を記述するが、セキュリティ上設定しない

radio

有 無
  • ラジオボタン
  • name プロパティは同じ名称にする
  • name をキーとして value の値が送信される
  • デフォルトで選択したい場合、checked="checked" を記述

checkbox

質問1 質問2 質問3
  • チェックボックス
  • name プロパティは統一する
  • チェックされた value の値が送信される
  • デフォルトで選択したい場合、checked="checked" を記述
  • [] を利用すると、データを連番の配列で送信可能

hidden

  • ブラウザ上には表示させないが、送信したい場合に利用
  • value の値が送信される

submit

デフォルト

CSS反映

  • form の実行ボタン
  • form 内の Formコントロールの値を送信

button

  • input type="submit" と同じ送信ボタン
  • 最近では input type="submit" より一般的に利用される
  • デフォルトは、type=submit
  • form の実行ボタン
  • form 内の Formコントロールの値を送信

textarea

  • 改行テキスト入力
  • name をキーとして 入力した値が value として送信される
  • 内容は textareaタグ内に記述
  • 行は rows で指定
  • 列は cols で指定

select

  • プルダウン
  • name 送信するキー
  • select タグ内に option を複数設定
  • option の value で値を設定
  • デフォルトで選択したい場合は、selected="selected"

label

  • ラベル
  • フォーム部品と項目名（ラベル）を関連付ける際に使用
  • for属性の値とフォームのid属性を関連づける
  • radio や checkbox ではラベル選択してチェックすることが可能
